Benefits of GitHub

GitHub offers several benefits, including:

  • Version Control
  • Collaboration
  • Code Hosting

Basic Commands

Here are some basic Git commands:

            git init
            git add .
            git commit -m "Initial commit"
            git push origin master
            git add file_name
            git pull origin branch_name

How to Push Your Project to GitHub

Follow these steps to push your project to GitHub:

  1. Create a new repository on GitHub.
  2. Initialize a Git repository locally using git init.
  3. Add and commit your files using git add and git commit.
  4. Set the remote repository using git remote add origin [repository URL].
  5. Push your changes to GitHub with git push origin master.
